| D-023 | 2026-03-13 | Phase 5 stores reviewed memory as hot references plus warm and cold retained payloads under the local fabric state root, keeps raw logs in a separate ephemeral store, and requires every retained memory change to be recorded with the frozen `MemoryPromotionDecision` shape. | Locks the Phase 5 memory path without changing the frozen Phase 2 interface field names or the runner command surface. |
